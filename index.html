Reflection on Lab 02: HTML Basics

For this lab, I practiced writing HTML by hand. I built a simple webpage with different tags like <header>, <main>, and <footer>. This was the first time I created an HTML document without using a template.

One challenge I faced was making sure all the tags were closed properly. I made a mistake at first by forgetting to close a tag, but I fixed it after testing the page and checking the code with the W3C validator.

What I Learned

  • How to use semantic HTML tags like <header>, <footer>, and <article>.
  • How to create ordered and unordered lists using <ul> and <ol>.
  • The importance of validating HTML to catch mistakes.

Steps I Took

  1. Created the HTML file in VS Code.
  2. Used tags like <header>, <article>, and <footer> to structure my page.
  3. Added headings, paragraphs, lists, and an external link.
  4. Tested my page in the browser and fixed errors after validation.

For help, I used FreeCodeCamp's interactive HTML tutorials, which made it easier to understand the syntax. You can check it out too: FreeCodeCamp HTML Basics
